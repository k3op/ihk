Tools, die das Leben eines IHK Prüfers leichter machen sollen.
Ich Prüfe Fachinformatiker für Systemintegration an der IHK Dortmund.

Die hier hinterlegten Tools setzen die Bewertungsbögen der [Handreichung der IHK zu Dortmund für IT-Berufe]([https://www.google.com](https://www.ihk.de/blueprint/servlet/resource/blob/6164116/2d46bfb195c7ee193395213625ae6dc8/it-handreichungen-stand-mai-2024-data.pdf)) sowie die Punktegewichtungen der Prüfungsordnung um.

Alle hinterlegten Tools basieren auf HTML mit Inline Javascript, sodass alle Berechnungen lokal im Browser ausgeführt werden und die Tools portabel und unabhängig von einem Webserver sind.
Zum ausführen einfach das gesamte Projekt oder das jeweilige Tool herunterladen und mit einem beliebigen Browser ausführen.
Alternativ sind die Tools auch unter [nacke.it/ihk](https://nacke.it/ihk) aufrufbar.
